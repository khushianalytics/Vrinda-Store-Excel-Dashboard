# Vrinda Store Sales Performance Dashboard – 2022

## Project Overview
This project is an interactive Excel dashboard created to analyze the annual sales performance of Vrinda Store for the year 2022.  

The dashboard helps track:
- Sales performance
- Order trends
- Customer demographics
- Sales channels
- Order delivery status
- State-wise revenue contribution
The dashboard is fully interactive using slicers and pivot charts.

## Tools & Features Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Data Cleaning
- Data Analysis
- Conditional Formatting
- Interactive Dashboard Design

---

## Dashboard Features

### Interactive Filters
Users can filter the dashboard using:
- Month
- Category
- Sales Channel

### KPI Metrics
The dashboard includes key performance indicators such as:
- Total Revenue
- Total Orders
- Delivery Success Rate
- Top Performing Channel

### Visual Analysis
The dashboard provides:
- Monthly Orders & Revenue Trend
- Sales Contribution by Gender
- Order Distribution by Age & Gender
- Orders by Sales Channel
- Top 5 States by Revenue
- Order Fulfillment Status


## Business Insights

### Sales Insights
- Women contributed a higher percentage of overall sales compared to men.
- Maharashtra generated the highest revenue among all states.
- Amazon was the top-performing sales channel.
- Most orders were successfully delivered.

### Customer Insights
- Adult women contributed significantly to total orders.
- Sales performance varied across different age groups.

### Operational Insights
- Delivery success rate was high.
- Cancellation and return percentages were comparatively low.
